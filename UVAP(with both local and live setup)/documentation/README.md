# University Automation System (UVAP)

## Overview
UVAP is a state-of-the-art University Automation System built with the MERN stack. It provides a complete solution for managing all university operations, from admissions to alumni.

## Features
- **Role-Based Access Control**: Admin, Student, Faculty, Finance, Exam, HR.
- **Academic Management**: Courses, Programs, Registration, Timetable.
- **Examination System**: Grading, Transcripts, Degree Audit.
- **Finance & HR**: Fee Collection, Payroll, Leave Management.
- **Advanced AI**: Sentiment Analysis for Teacher Reviews.
- **Dual Version**:
    - `local_version_uvap`: For development and testing.
    - `live_version_uvap`: Production-ready with Docker and Nginx.

## Directory Structure
- `local_version_uvap/`: Contains the development version.
- `live_version_uvap/`: Contains the production version.
- `documentation/`: Detailed project documentation.
